Instalando Git e GitHub

Capítulo 18: aula 03

11 de Novembro de 2024

Definições:

Na aula 00 (zero) foi apresentado o conteúdo do curso assim como o objetivo e na aula 1 (um) o professor demostrou a necessidade da utilização do GitHub assim como suas funcionalidades. Foi mostradado que um bom desenvolvedor que faz uso de tal recurso terá um alto nível de produtividade assim como a possibilidade de corrigir seu projeto caso ele apresente um resuldado insatifatório.

O capítulo 3 (três) íra falar sobre VERSIONAMENTO, REPOSITÓRIOS LOCAIS e REPOSITÓRIOS REMOTOS.

O sistema adotado foi o LINUX Ubuntu e para que o sistema de Versionamento possa funcionar será preciso instalar primeiramente o Git com o seguinte comando no terminal: 

    sudo apt-get install git-all

Logo em seguida recomenda-se instalar o GitHub Desktop por motivos didáticos, existem outros programas que desempenha as mesmas funcionalidades, porém, esse poderá ser usado de modo mais intuitivo.

O segundo passo será criar uma conta no GitHub. É importante lembrar que o NOME e o EMAIL sejam interessante para a comunidade programadores e/ou possíveis impregadores.
